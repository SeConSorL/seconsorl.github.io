用法

HTML 中的 Javascript 脚本代码必须位于 <script> 与 </script> 标签之间。脚本可位于 HTML 的 <body> 或 <head> 部分中，或者同时存在于两个部分中。

可以在 HTML 文档中放入不限数量的脚本。

通常的做法是把函数放入 <head> 部分中，或者放在页面底部。这样就可以把它们安置到同一处位置，不会干扰页面的内容。

外部的 JavaScript

可以把脚本保存到外部文件中。外部文件通常包含被多个网页使用的代码。外部 JavaScript 文件的文件扩展名是 .js。

如需使用外部文件，请在 <script> 标签的 src 属性中设置该 .js 文件的路径。

外部js文件不使用 <script>标签，直接写 js 代码。

在Chrome中查看

1、Console 窗口调试 JavaScript 代码

F12的Console 窗口（控制台）调试 JavaScript代码，在 > 符号后输入我们要执行的代码。

点击禁止的图标，可以清空 Console 窗口的内容

2、Chrome snippets 小脚本

在 Sources （源）面板，选择 Snippets （代码段）选项卡可以创建脚本文件

输出/显示

JavaScript 可以通过不同的方式来输出数据：

  • 使用 window.alert() 弹出警告框。
  • 使用 document.write() 方法将内容写到 HTML 文档中。
  • 使用 innerHTML 写入到 HTML 元素。
  • 使用 console.log() 写入到浏览器的控制台。

document.write() //写到 HTML 文档

HTML 输出流中使用 document.write，相当于添加在原有html代码中添加一串html代码。

而如果在文档加载后使用（如使用函数），会覆盖整个文档。

原因：如果在写之前没有调用document.open, 浏览器会自动调用open。每次写完关闭之后重新调用该函数，会导致页面被重写。

alert()/window.alert()

alert() 函数在 JavaScript 中并不常用，但它对于代码测试非常方便。

括号里面添加数据，使用的时候页面会出现警告框，提示内容即为括号内容。

查找/修改 HTML 元素

如需从 JavaScript 访问某个 HTML 元素，您可以使用 document.getElementById(id) 等方法。

innerHTML

是DOM（文档对象模型）页面元素的一个属性，可以获取或插入元素内容。

还有其他的属性如innerText、innerElement等，可以精确到某一个具体的元素来进行更改。

console.log() //写到控制台

用法类似alert()。可以添加显示样式。可以像 C 语言用 % 来格式化输出变量

console.log()的用处

主要是方便调试javascript用的, 可以看到你在页面中输出的内容。

他能看到结构化的东西，如果是alert，弹出一个对象就是[object object]，但是console能看到对象的内容。

console不会打断你页面的操作，如果用alert弹出来内容，那么页面就死了，但是console输出内容后你页面还可以正常操作。

语法

字面量

在编程语言中，一般固定值称为字面量。

  • 数字（Number）

3.14、1001、123e5（显示为12300000）

  • 字符串（String）

可以使用单引号或双引号

  • 表达式

表达式计算

  • 数组（Array）

[ ]里面添加数据，显示时不会显示**[]**

  • 对象（Object）

{ }字典里加键值对，{firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}

内部定义成员时，类似PASCAL，成员名+:+具体内容

  • 函数（Function）

比如function myFunction(a, b) { return a * b;}

使用时要添加 () 调用，否则会认为是对象的属性，返回定义函数的字符串

变量

使用关键字 var 来定义变量， 使用等号来为变量赋值。可以重复定义同一个变量名，但是只是把原变量去掉，加入一个新变量。

如果把值赋给尚未声明的变量，该变量将被自动作为 window 的一个属性。非严格模式下，它是全局对象的可配置属性，可以删除。

变量可以通过变量名访问。

拥有动态类型。这意味着相同的变量可用作不同的类型（Python）

当声明新变量时，可以使用关键词 “new” 来声明其类型：如var carname=new String;

操作符

使用算术运算符来计算值，\+ - * /等

使用赋值运算符给变量赋值，=

使用比较运算符来判断逻辑，== != < >

JavaScript 是脚本语言，浏览器会在读取代码时，逐行地执行脚本代码。而对于传统编程来说，会在执行前对所有代码进行编译。

换行

可以在文本字符串中使用反斜杠\对代码行进行换行

注意：不能破坏变量、函数等的整体性

在保证单个整体性时，可以换行，比如定义对象时。

注释

单行：

双斜杠 // 后的内容将会被浏览器忽略

多行：

以 /* 开始，以 */ 结尾

如果用户不能确定浏览器是否支持JavaScript脚本，那么可以应用HTML提供的注释符号进行验证。以 <-- 开始以 --> 结束

如果浏览器支持JavaScript脚本，则加单行或多行注释会起作用。

数据类型

数字，字符串，数组，对象等等，可以通过字面量赋值。

Symbol 是 ES6 引入了一种新的原始数据类型，表示独一无二的值。

使用规则同C

可以使用 typeof 操作符来查看类型。

Undefined 和 Null

Undefined 表示变量不含有值，即未赋值

可以通过将变量的值设置为 null 来清空变量。

函数

函数可以重复引用。引用一个函数 = 调用函数(执行函数内的语句)。

语法，我的理解是PASCAL加C。使用PASCAL里的函数关键字function，其余同C的语法。

function myFunction(var1,var2) //传参时，按顺序
{
//代码
}

可以加return，但是不是必须的，因为它也没有规定返回值的类型等。

ES6 新增箭头函数=>，在使用时可以省略 function 关键字，且当只有一个参数时可以把括号()去掉。

大小写

JavaScript 对大小写是敏感的。

使用 Unicode 字符集。
