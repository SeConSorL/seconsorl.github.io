行内元素和块元素

  • 块元素：会另起一个行。无论内容多少，该元素会独占一行。
  • 行内元素：内容撑开宽度，左右都是行元素的可以排成一行。

nav

<nav>标签定义导航链接的部分。

并不是所有的 HTML 文档都要使用到<nav> 元素。<nav>元素只是作为标注一个导航链接的区域。

在不同设备上（手机或者PC）可以制定导航链接是否显示，以适应不同屏幕的需求。

图像标签

<img src="路径" alt="图片名字" title="悬停显示文字" width="300" height="300">
  • src：图片路径，../表示上一级，./表示当前目录下，（必填）
  • alt：当src路径失效，显示该内容，（必填）
  • title：鼠标悬停时，显示内容
  • width、height：图片的宽度和高度

链接标签

页面间链接

<a href="path" target="目标窗口位置">文本或图像</a>
  • href：链接路径（必填）
  • target：表示窗口在哪里打开
    • _blank：新窗口打开
    • _self：在当前网页打开（默认）

锚链接

<a name="top"> 顶部 </a>
<a href="#top"> 跳转到顶部 </a>
<a href="另一个HTML文件路径+#{name}"> 跳转到另一个页面的某一个标签处 </a>

表单元素格式

属性 说明
type 指定元素的类型。text、password、checkbox（多选框）、radio（单选框）、submit（提交）、reset（清空内容）、file、hidden、image（图像按钮）和button（普通按钮）,默认为text
name 指定表单元素的名称
value 元素的初始值。type为radio时必须指定一个值
size 指定表单元素的初始宽度。当type为text或password时，表单元素的大小以字符为单位。对于其他类型，宽度以像素为单位
maxlength type为text或password时，输入的最大字符数
checked type为radio或checkbox时，指定按钮是否是被选中

点击image类型也有sumit的效果
