主要功能与界面介绍

1、最上方是菜单栏

2、左侧导航栏（包含数据库管理、分组、在线检索）

3、中间文献总览（上部为检索工具栏，下部为文献列表）

4、所选择的文献的详细信息

左侧导航栏

  • 同步设置（Synv Configuration）
  • 显示所有的参考文献（All References）
  • 最近添加（Recently Added）：右键可以选择"最近多少天"
  • 未分类的文献（Unfiled）
  • 回收站（Trash）：记录最近删除的文献
  • 我的分组（MY GROUPS）：显示所有定义的组别（包括简单分组、组集、智能分组）
  • 我的标签（MY TAGS）：可以自定义标签，按照标签分类
  • 查找全文（FIND FULL TEXT）：查找本地库里含有全文的文献
  • 在线检索（ONLINE SEARCH）：可以在多个数据库在线检索文献

最下方有搜索框，搜索分组

文献总览

检索框

  • 分为Simple和Advance两种，和一般的文献检索相同

  • 下方右侧有快捷工具栏，分别实现手动添加文献、添加好友（可以分享library）、输出library、查找全文、输出一个选中的文献Web引用报告

  • 此外还有**搜索选项（Search options）**可以实现进阶操作

文献列表

  • 最上方的文献信息标题栏：右键可添加更多的标题
  • 有回形针标识表示已有附件
  • 文献的重要程度（Rating）：可通过加星来体现文献是否重要
  • 标记笔记（reseach note）：作信息补充
  • 文献显示格式内容：可调整
文献的已读状态

当在文献列表区往下滑动阅读文献时，点击过某篇文献就会显示已读状态，但有时软件默认不会显示，这时需要进行设置，操作步骤如下

  • 点击"Edit"
  • 选择"Preferences"
  • 在弹出对话框中点击"Read/UnreacT，勾选第一个选框，点击确定

数据库

建立

点击"File" -> “new”

选择文件位置，创建enl文件

此时会产生个人数据库文件，包括MyEndNoteLibrary.Data文件夹和MyEndNoteLibrary.en文件

MyEndNoteLibrary.Data文件夹中包括三个子文件夹：PDF、rdb、tdb

PDF子文件夹主要用于存放下载的文献全文，其它两个文件用于存放数据

打开

点击"File" -> “open” 选择enl文件即可

此时，软件窗口的标题栏就会显示当前数据库的名字 之后的操作默认是在该数据库进行的

添加文献

本地导入

点击"File" -> “import” 选择"file"或者"folder"

在弹出的对话框中选择"choose" 来选择文件

本方法是通过在线检索PDF文件的DOI号从而在线下载文献的各种信息

中文文献或者没有DOI号的文献需要从相应数据库下载导入EndNote

在线导入

在Online search 中选择要用的数据库。若没有则点击➕添加数据库

通过进行检索，找到文献

检索结果出来后，点击➕，将文献添加进库中

网站导入

在一些提供科研论文检索的网站中，点击"引用（Cite）" 或者"导出（Export）" 会得到txt文件

通过本地导入 在"import option“中选择Endnote import 再导入txt文件即可

网站输出

以Pubmed为例：

  • 进入Pubmed主页 https://pubmed.ncbi.nlm.nih.gov/

  • 在检索框通过关键词进行检索勾选需要导出的文献（可单个或多个）

  • 点击"send to "

  • 在下拉菜单里点击"citation manager "

  • 选择导出范围

    All results on this page 本页全部文献

    Allresults 所有的文献

    Selection 已选中文献

手动添加

点击主界面"Reference"里面的"New Reference"或者点击快捷工具栏里的添加新文献

手动设置引用类型、作者、题目等内容

文件夹关联同步

  • 点击"Edit” -> “Preferences”
  • 在弹出对话框中点击"PDF Handling"，选择要同步的文件夹，勾选自动导入

数据库共享

想要实现单机版（EndNote在电脑软件中的数据库）与在线版（网页版）的同步或将数据库分享给好友

可点击左侧导航栏的"SyncConfiguration"或者快捷工具栏的"Sharethisgroup"同步与共享功能

需要登录EndNote账号

点击左上角"Edit" -> “Preference” -> “Sync” 在 “EndNote Account” Credentials 处输入账号密码即可。

同步功能在线版可以同步到单机版，但是单机版不能同步到在线版

实现单机版同步到网页版

  • 上方菜单栏tools中Output Styles选项
  • 选择输出格式为EndNote Export
  • 主界面左上角点击File
  • 选择输出Export
  • 更改要输出的文件名
  • 在Output Styles里下拉选择输出格式为EndNote Export

数据库备份

“File” -> “Compress Library”
